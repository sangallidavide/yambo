SNESMonitorFields

 Monitors the residual for each field separately

Synopsis

#include "petscdmplex.h" 
#include "petscsnes.h"   
PetscErrorCode SNESMonitorFields(SNES snes, PetscInt its, PetscReal fgnorm, PetscViewerAndFormat *vf)
Collective on SNES

Input Parameters

snes - the SNES context
its - iteration number
fgnorm - 2-norm of residual
vf - PetscViewerAndFormat of type ASCII

Notes

This routine prints the residual norm at each iteration.

Keywords

SNES, nonlinear, default, monitor, norm

See Also

SNESMonitorSet(), SNESMonitorDefault()
