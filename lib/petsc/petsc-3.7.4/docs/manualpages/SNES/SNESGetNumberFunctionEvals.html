SNESGetNumberFunctionEvals

 Gets the number of user provided function evaluations done by SNES.

Synopsis

#include "petscsnes.h"  
PetscErrorCode  SNESGetNumberFunctionEvals(SNES snes, PetscInt *nfuncs)
Not Collective

Input Parameter

snes -SNES context

Output Parameter

nfuncs -number of evaluations

Notes: Reset every time SNESSolve is called unless SNESSetCountersReset() is used.

Keywords

SNES, nonlinear, get, maximum, unsuccessful, steps
