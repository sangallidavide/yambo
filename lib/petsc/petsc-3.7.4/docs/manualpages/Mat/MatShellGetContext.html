MatShellGetContext

 Returns the user-provided context associated with a shell matrix.

Synopsis

#include "petscmat.h" 
PetscErrorCode  MatShellGetContext(Mat mat,void *ctx)
Not Collective

Input Parameter

mat -the matrix, should have been created with MatCreateShell()

Output Parameter

ctx -the user provided context

Fortran Notes: To use this from Fortran you must write a Fortran interface definition for this function that tells Fortran the Fortran derived data type that you are passing in as the ctx argument.

Keywords

matrix, shell, get, context

See Also

MatCreateShell(), MatShellSetOperation(), MatShellSetContext()
