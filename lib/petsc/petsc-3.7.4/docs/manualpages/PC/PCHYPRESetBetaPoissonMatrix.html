PCHYPRESetBetaPoissonMatrix

 Set Poisson matrix

Synopsis

#include "petscpc.h" 
PetscErrorCode PCHYPRESetBetaPoissonMatrix(PC pc, Mat A)
Collective on PC

Input Parameters

pc - the preconditioning context
A - the matrix

Notes: A should be obtained by discretizing the Poisson problem with linear finite elements. Following HYPRE convention, the scalar Poisson solver of AMS can be turned off by passing NULL.
