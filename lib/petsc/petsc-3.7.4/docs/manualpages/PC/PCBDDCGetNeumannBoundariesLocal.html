PCBDDCGetNeumannBoundariesLocal

 Get parallel IS for Neumann boundaries (in local ordering)

Synopsis

#include "petscpc.h" 
PetscErrorCode PCBDDCGetNeumannBoundariesLocal(PC pc,IS *NeumannBoundaries)
Collective

Input Parameters

pc -the preconditioning context

Output Parameters

NeumannBoundaries -index set defining the subdomain part of Neumann boundaries

Notes

The IS returned could be the same passed in earlier by the user (if provided with PCBDDCSetNeumannBoundariesLocal) or a global-to-local map of the global IS (if provided with PCBDDCSetNeumannBoundaries). In the latter case, the IS will be available after PCSetUp.
