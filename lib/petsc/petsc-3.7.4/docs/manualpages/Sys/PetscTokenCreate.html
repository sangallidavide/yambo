PetscTokenCreate

 Creates a PetscToken used to find tokens in a string

Synopsis

#include "petscsys.h"   
PetscErrorCode  PetscTokenCreate(const char a[],const char b,PetscToken *t)
Not Collective

Input Parameters

string - the string to look in
b - the separator character

Output Parameter

t -the token object

Notes

This version is different from the system version in that it allows you to pass a read-only string into the function.

Not for use in Fortran

See Also

PetscTokenFind(), PetscTokenDestroy()
