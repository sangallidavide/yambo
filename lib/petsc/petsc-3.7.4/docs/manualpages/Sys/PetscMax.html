PetscMax

 Returns maxium of two numbers

Synopsis

#include <petscmath.h>
type max PetscMax(type v1,type v2)
Not Collective

Input Parameter

v1 - first value to find maximum of
v2 - second value to find maximum of

Notes: type can be integer or floating point value
