PetscIsNormalReal

 Returns PETSC_TRUE if the input value satisfies isnormal()

Synopsis

#if defined(PETSC_USE_REAL___FLOAT128)
PetscBool PetscIsNormalReal(PetscReal a)

Input Parameter

a -the PetscReal Value

Notes: uses the C99 standard isnormal() on systems where they exist. Uses isnormalq() with __float128 Otherwises always returns true
