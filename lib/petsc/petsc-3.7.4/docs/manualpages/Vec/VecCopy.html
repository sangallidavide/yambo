VecCopy

 Copies a vector. y <- x

Synopsis

#include "petscvec.h"   
PetscErrorCode  VecCopy(Vec x,Vec y)
Logically Collective on Vec

Input Parameter

x -the vector

Output Parameter

y -the copy

Notes

For default parallel PETSc vectors, both x and y must be distributed in the same manner; local copies are done.
