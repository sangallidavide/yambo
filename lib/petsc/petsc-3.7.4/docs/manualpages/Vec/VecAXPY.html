VecAXPY

 Computes y = alpha x + y.

Synopsis

#include "petscvec.h"   
PetscErrorCode  VecAXPY(Vec y,PetscScalar alpha,Vec x)
Logically Collective on Vec

Input Parameters

alpha - the scalar
x, y - the vectors

Output Parameter

y -output vector

Notes: x and y MUST be different vectors

See Also

VecAYPX(), VecMAXPY(), VecWAXPY()
