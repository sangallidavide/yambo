VecScatterView

 Views a vector scatter context.

Synopsis

#include "petscvec.h"    
PetscErrorCode  VecScatterView(VecScatter ctx,PetscViewer viewer)
Collective on VecScatter

Input Parameters

ctx - the scatter context
viewer - the viewer for displaying the context
