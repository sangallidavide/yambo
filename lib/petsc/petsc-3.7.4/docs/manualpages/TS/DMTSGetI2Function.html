DMTSGetI2Function

 get TS implicit residual evaluation function for 2nd order systems

Synopsis

#include "petscts.h" 
PetscErrorCode DMTSGetI2Function(DM dm,TSI2Function *fun,void **ctx)
Not Collective

Input Argument

dm -DM to be used with TS

Output Arguments

fun - function evaluation function, see TSSetI2Function() for calling sequence
ctx - context for residual evaluation

Note

TSGetI2Function() is normally used, but it calls this function internally because the user context is actually associated with the DM.

See Also

DMTSSetI2Function(),TSGetI2Function()
