TSAdjointSetSteps

 Sets the number of steps the adjoint solver should take backward in time

Synopsis

#include "petscts.h"  
PetscErrorCode  TSAdjointSetSteps(TS ts,PetscInt steps)
Logically Collective on TS

Input Parameters

Notes: Normally one does not call this and TSAdjointSolve() integrates back to the original timestep. One can call this so as to integrate back to less than the original timestep

Keywords

TS, timestep, set, maximum, iterations
