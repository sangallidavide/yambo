TSAdaptInitializePackage

 This function initializes everything in the TSAdapt package. It is called from PetscDLLibraryRegister() when using dynamic libraries, and on the first call to TSCreate_GL() when using static libraries.

Synopsis

#include "petscts.h" 
PetscErrorCode  TSAdaptInitializePackage(void)

Keywords

TSAdapt, initialize, package
