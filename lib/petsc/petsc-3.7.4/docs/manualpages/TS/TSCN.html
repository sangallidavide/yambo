TSCN

 ODE solver using the implicit Crank-Nicolson method.

Notes

TSCN is equivalent to TSTHETA with Theta=0.5 and the "endpoint" option set. I.e. 

 -ts_type theta -ts_theta_theta 0.5 -ts_theta_endpoint
