KSPSetApplicationContext

 Sets the optional user-defined context for the linear solver.

Synopsis

#include "petscksp.h" 
PetscErrorCode  KSPSetApplicationContext(KSP ksp,void *usrP)
Logically Collective on KSP

Input Parameters

ksp - the KSP context
usrP - optional user context

Fortran Notes: To use this from Fortran you must write a Fortran interface definition for this function that tells Fortran the Fortran derived data type that you are passing in as the ctx argument.

Keywords

KSP, set, application, context
