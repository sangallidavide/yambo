KSPPIPEFCGGetMmax

 get the maximum number of previous directions PIPEFCG will store

Synopsis

#include "petscksp.h"  
PetscErrorCode KSPPIPEFCGGetMmax(KSP ksp,PetscInt *mmax)
Note: PIPEFCG stores mmax+1 directions at most (mmax previous ones, and the current one)

Not Collective

Input Parameter

ksp -the Krylov space context

Output Parameter

mmax -the maximum number of previous directons allowed for orthogonalization

Options Database

-ksp_pipefcg_mmax <N> -

Keywords

KSP, PIPEFCG, truncation
