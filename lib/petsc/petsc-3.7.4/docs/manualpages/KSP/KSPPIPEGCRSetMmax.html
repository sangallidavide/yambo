KSPPIPEGCRSetMmax

 set the maximum number of previous directions PIPEGCR will store for orthogonalization

Synopsis

#include "petscksp.h"  
PetscErrorCode KSPPIPEGCRSetMmax(KSP ksp,PetscInt mmax)
Note: mmax + 1 directions are stored (mmax previous ones along with a current one) and whether all are used in each iteration also depends on the truncation strategy (see KSPPIPEGCRSetTruncationType)

Logically Collective on KSP

Input Parameters

ksp - the Krylov space context
mmax - the maximum number of previous directions to orthogonalize againt

Options Database

-ksp_pipegcr_mmax <N> -
