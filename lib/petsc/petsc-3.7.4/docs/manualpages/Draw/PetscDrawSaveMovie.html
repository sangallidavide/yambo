PetscDrawSaveMovie

 Saves a movie from previously saved images

Synopsis

#include "petscdraw.h" 
PetscErrorCode PetscDrawSaveMovie(PetscDraw draw)
Collective on PetscDraw

Input Parameters

draw -the drawing context

Notes: this is not normally called by the user. The ffmpeg utility must be in your path to make the movie.
