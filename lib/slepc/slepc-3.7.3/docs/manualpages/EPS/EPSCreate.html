EPSCreate

Creates the default EPS context.

Synopsis

#include "slepceps.h" 
PetscErrorCode EPSCreate(MPI_Comm comm,EPS *outeps)
Collective on MPI_Comm

Input Parameter

comm  - MPI communicator

Output Parameter

eps  - location to put the EPS context

Note

The default EPS type is EPSKRYLOVSCHUR
