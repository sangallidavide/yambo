VecCompSetSubVecs

Resets the number of subvectors defining a compound vector, of replaces the subvectors.

Synopsis

#include "slepcvec.h" 
PetscErrorCode VecCompSetSubVecs(Vec win,PetscInt n,Vec *x)
Collective on Vec

Input Parameters

win  - compound vector
n  - number of child vectors
x  - array of child vectors
